Visão profissional

Meu GitHub como currículo técnico em construção.

Em vez de manter os códigos soltos, organizei este repositório para servir como uma apresentação profissional. A raiz virou uma página de portfólio, e cada projeto passou a ter um espaço próprio para navegação, leitura e futura expansão.

O objetivo é que recrutadores, professores e empresas consigam entender rapidamente o que eu já construí, quais tecnologias estou praticando e como minha evolução acontece na prática. Aqui, estudo vira evidência.
