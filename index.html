Sobre mim

Quem eu sou e o que busco construir com este GitHub.

Meu nome é Gabriel Belarmino Martins. Concluí o ensino médio na Escola Técnica Estadual Doutor Júlio Cardoso e atualmente sigo em formação na Uni-FACEF, em Franca, São Paulo. Tenho usado meus projetos para mostrar evolução prática, não apenas interesse teórico.

Este perfil foi pensado para reunir trabalhos acadêmicos, exercícios de lógica e experiências em desenvolvimento web de forma mais profissional. Quero que cada repositório mostre organização, clareza, consistência e potencial de crescimento técnico.

Visão profissional

Meu GitHub como currículo técnico em construção.

Em vez de manter os códigos soltos, organizei este repositório para servir como uma apresentação profissional. A raiz virou uma página de portfólio, e cada projeto passou a ter um espaço próprio para navegação, leitura e futura expansão.

O objetivo é que recrutadores, professores e empresas consigam entender rapidamente o que eu já construí, quais tecnologias estou praticando e como minha evolução acontece na prática. Aqui, estudo vira evidência.
