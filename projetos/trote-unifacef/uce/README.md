# UCE - Trote Solidário Uni-FACEF 2026

Este diretório reúne materiais acadêmicos ligados ao projeto do `Trote Solidário Uni-FACEF 2026`, desenvolvido na Unidade Curricular de Extensão (UCE) de Fundamentos de Programação e Desenvolvimento Web.

- `RELATORIO-PARCIAL-UCE.md`: versão adaptada para leitura direta no GitHub.
- `RELATORIO-PARCIAL-UCE.docx`: arquivo original do relatório.

## Síntese do conteúdo

O relatório descreve a participação da Equipe Verde na UCE de Fundamentos de Programação e Desenvolvimento Web, conectando o aprendizado de HTML, CSS e JavaScript ao desenvolvimento de um site para o `Trote Solidário Uni-FACEF 2026`.

Entre os principais pontos apresentados no documento estão:

- o objetivo de transformar o regulamento do evento em uma experiência digital mais clara para os participantes;
- a análise das provas, metas e critérios de pontuação do Trote Solidário;
- a ação social realizada em `19 de março de 2026` na `NV Sociedade Solidária`, em Franca-SP;
- a participação da equipe na `Noite de Encerramento`, em `27 de março de 2026`;
- o desenvolvimento da calculadora em JavaScript para simular a pontuação da gincana;
- as contribuições técnicas e sociais da UCE para a formação dos estudantes.

O texto também relaciona o projeto aos Objetivos de Desenvolvimento Sustentável da ONU, com destaque para:

- `ODS 1`: Erradicação da Pobreza
- `ODS 9`: Indústria, Inovação e Infraestrutura
- `ODS 10`: Redução das Desigualdades

## Observação sobre o CSS

O arquivo `style.css` foi preservado aqui como material de referência da versão acadêmica documentada no relatório. Ele não substitui automaticamente o CSS atualmente usado na página publicada do projeto, que continua organizado separadamente dentro do repositório.
