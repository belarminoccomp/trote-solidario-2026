Competências desenvolvidas

Durante a realização desta atividade, foram trabalhadas as seguintes competências:

Controle de fluxo;
Uso de contadores e acumuladores;
Validação de entradas de dados;
Estruturas de repetição com condições de parada;
Organização lógica do raciocínio;
Interpretação de enunciados e complementação de código.
Importância desta etapa na trilha de aprendizagem

Este material representa um dos primeiros marcos formais da minha evolução em JavaScript ao longo do curso. Por esse motivo, foi preservado como parte da linha do tempo da disciplina, registrando o desenvolvimento inicial de conceitos fundamentais de programação e lógica computacional.
