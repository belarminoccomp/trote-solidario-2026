Objetivo

A proposta desta atividade foi desenvolver a lógica de programação utilizando estruturas de repetição e controle de fluxo em JavaScript, reforçando conceitos fundamentais da linguagem.

Entre os principais objetivos trabalhados estão:

praticar estruturas de repetição;
compreender regras de parada;
utilizar comandos de controle;
desenvolver raciocínio lógico;
interpretar problemas computacionais;
organizar soluções em código.
