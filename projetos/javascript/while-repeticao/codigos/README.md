# Códigos da Atividade de `while`

Esta pasta reúne as soluções da atividade de estruturas de repetição em arquivos `.js`, preservando o conteúdo prático da disciplina em formato de código real dentro do repositório.

## Organização

Os arquivos foram nomeados de forma sequencial, com títulos descritivos, para facilitar leitura, revisão e comparação:

- `ex01-contador-crescente.js`
- `ex02-contador-decrescente.js`
- `ex03-soma-dos-primeiros-numeros.js`
- `ex04-pares-com-continue.js`
- `ex05-adivinhe-o-numero.js`
- `ex06-validacao-de-nota.js`
- `ex07-tabuada-interativa.js`
- `ex08-senha-com-limite-de-tentativas.js`
- `ex09-calculadora-acumulada.js`
- `ex10-numero-primo.js`
- `ex11-menu-de-operacoes.js`
- `ex12-fizzbuzz-com-while.js`

## Critério de preservação

As soluções desta pasta foram organizadas com base no conteúdo da atividade original em Word. Nos exercícios interativos, foi mantido o uso de `prompt()`, pois esse era o formato previsto no material, voltado à execução em navegador ou em ambientes compatíveis com entrada interativa.

## Finalidade no portfólio

Mais do que registrar uma entrega acadêmica, esta pasta funciona como um ponto de consulta da minha evolução em JavaScript. Assim, o conteúdo permanece acessível para revisão, comparação com etapas futuras e documentação da minha linha do tempo técnica.
